# Python-quiz


## This is a simple python code using modular code and functions.
## This quiz asks the participant a series of "Fun fact" questions and calculates a final score.


## How to run the quiz: 

### Requirements
- python 3 installed on computer


## Running the program:
1. Download or clone repositry,
2. Open a terminal in project folder.
3. Run the following command:

'''bash
Python3 quiz.py



''' Markdown:
## How the quiz works:
- - Questions are stored in a list called 'quiz', each as a tuple:
  (question string, answer string or list of possible answers)
- The 'run_quiz(Questions)' function:
  - Loops through all questions
  - Compares user input to the correct answer(s)
  - Supports multiple valid answers
  - Tracks and displays the final score
